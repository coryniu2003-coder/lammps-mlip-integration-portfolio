# MatterSim integration

This example requests `MatterSim-v1.0.0-1M` through ML-GNNP.

```bash
bash scripts/doctor.sh --model mattersim
bash scripts/run_demo.sh --model mattersim
```

The development setup used `mattersim 1.2.0` in the `mlff_mattersim_orb` environment. Model weights may download on first use.
